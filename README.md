# 🧊 Neon Markdown Visualizer

A sleek, glassmorphism-inspired Markdown editor and previewer built for the browser.

Designed with a neon blue aesthetic, this tool provides a clean split interface to write Markdown on one side and instantly visualize the rendered output on the other.

---

## ✨ Features

* 🖥️ **Split Editor / Preview Layout**
  Write Markdown on the left, preview the result on the right.

* 📱 **Responsive Design**
  Automatically stacks vertically on smaller screens.

* 🔢 **Live Line Numbers**
  IDE-like editing experience with synchronized scrolling.

* 📂 **Open `.md` Files**
  Load any local Markdown file directly into the editor.

* 💾 **Download Markdown**
  Export your work instantly as a `.md` file.

* ⚡ **Manual Render Button**
  Control when your Markdown is rendered with the *Visualize* button.

* 🎨 **Neon Glass UI**
  Modern blue-glass aesthetic with subtle glow effects.

## 📌 How It Works

* Write Markdown inside the editor panel
* Click **Visualize** to render it
* Use:

  * **Open** → load a `.md` file
  * **Download** → save your content

The Markdown parser is implemented manually in JavaScript, supporting:

* Headings
* Lists (ordered & unordered)
* Code blocks
* Inline formatting (bold, italic, code)
* Links & images
* Blockquotes
* Tables
